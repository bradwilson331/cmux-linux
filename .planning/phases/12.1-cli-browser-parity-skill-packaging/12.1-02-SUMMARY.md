## Accomplishments
- Added browser_surface_counter and browser_surface_refs to AppState for ephemeral surface handle tracking
- Updated BrowserOpen handler to allocate surface:N refs and return them in response alongside UUID
- Added browser.list handler returning all registered surfaces with ref, uuid, and status
- Added resolve_surface_ref helper that handles both surface:N and raw UUID formats
- BrowserAction proxy now resolves surface_ref to UUID before forwarding to agent-browser daemon
- Stale surface ref returns structured error with code "surface_not_found" and available refs list
