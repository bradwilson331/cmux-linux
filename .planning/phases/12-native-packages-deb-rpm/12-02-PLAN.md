**Part A: Create `packaging/rpm/cmux.spec`** (per D-02)

This is the RPM spec file. Version is injected via `--define` at build time (per D-03 -- no build steps in spec).

```spec
Name:           cmux
Version:        %{_cmux_version}
Release:        1%{?dist}
Summary:        GPU-accelerated terminal multiplexer
License:        Proprietary
URL:            https://cmux.dev

# Pre-built binary package -- no Source0, no %build
AutoReqProv:    no

Requires:       gtk4
Requires:       fontconfig
Requires:       freetype
Requires:       oniguruma
Requires:       mesa-libGL
Requires:       mesa-libEGL
Requires:       harfbuzz
Requires:       glib2
Requires:       cairo
Requires:       pango
Requires:       libepoxy
Requires:       libxkbcommon
Requires:       graphene

%description
cmux is a GPU-accelerated terminal with tabs, splits, workspaces,
and socket CLI control -- powered by Ghostty.

%install
install -Dm0755 %{_sourcedir}/cmux-app %{buildroot}%{_bindir}/cmux-app
install -Dm0755 %{_sourcedir}/cmux %{buildroot}%{_bindir}/cmux
install -Dm0755 %{_sourcedir}/cmuxd-remote %{buildroot}%{_libdir}/cmux/cmuxd-remote

install -Dm0644 %{_sourcedir}/com.cmux_lx.terminal.desktop %{buildroot}%{_datadir}/applications/com.cmux_lx.terminal.desktop
install -Dm0644 %{_sourcedir}/com.cmux_lx.terminal.metainfo.xml %{buildroot}%{_datadir}/metainfo/com.cmux_lx.terminal.metainfo.xml

install -Dm0644 %{_sourcedir}/icons/48x48.png %{buildroot}%{_datadir}/icons/hicolor/48x48/apps/com.cmux_lx.terminal.png
install -Dm0644 %{_sourcedir}/icons/128x128.png %{buildroot}%{_datadir}/icons/hicolor/128x128/apps/com.cmux_lx.terminal.png
install -Dm0644 %{_sourcedir}/icons/256x256.png %{buildroot}%{_datadir}/icons/hicolor/256x256/apps/com.cmux_lx.terminal.png

install -Dm0644 %{_sourcedir}/cmux.bash %{buildroot}%{_datadir}/bash-completion/completions/cmux
install -Dm0644 %{_sourcedir}/_cmux %{buildroot}%{_datadir}/zsh/vendor-completions/_cmux
install -Dm0644 %{_sourcedir}/cmux.fish %{buildroot}%{_datadir}/fish/vendor_completions.d/cmux.fish

install -Dm0644 %{_sourcedir}/cmux.1.gz %{buildroot}%{_mandir}/man1/cmux.1.gz

%files
%{_bindir}/cmux-app
%{_bindir}/cmux
%{_libdir}/cmux/cmuxd-remote
%{_datadir}/applications/com.cmux_lx.terminal.desktop
%{_datadir}/metainfo/com.cmux_lx.terminal.metainfo.xml
%{_datadir}/icons/hicolor/48x48/apps/com.cmux_lx.terminal.png
%{_datadir}/icons/hicolor/128x128/apps/com.cmux_lx.terminal.png
%{_datadir}/icons/hicolor/256x256/apps/com.cmux_lx.terminal.png
%{_datadir}/bash-completion/completions/cmux
%{_datadir}/zsh/vendor-completions/_cmux
%{_datadir}/fish/vendor_completions.d/cmux.fish
%{_mandir}/man1/cmux.1.gz
```

Key decisions in the spec:
- `AutoReqProv: no` -- disables automatic dependency detection since we're packaging pre-built binaries and declaring deps explicitly (per D-08, D-10)
- Requires list from FEDORA_FALLBACK table in detect-deps.sh (per D-10, D-11)
- `%{_libdir}` expands to `/usr/lib64` on Fedora x86_64 (standard RPM macro)
- No `%build` section -- binaries are pre-built (per D-03)
- Source files are staged into `_sourcedir` by the build script before rpmbuild runs

**Part B: Create `packaging/scripts/build-rpm.sh`** (per D-02)

Script structure:
1. Shebang: `#!/usr/bin/env bash` + `set -euo pipefail`
2. SCRIPT_DIR/REPO_ROOT pattern
3. Check rpmbuild is installed: `command -v rpmbuild &>/dev/null || { echo "ERROR: rpmbuild not found. Install with: sudo apt install rpm" >&2; exit 1; }`
4. Parse binary paths (per D-04):
   - `CMUX_APP="${1:-$REPO_ROOT/target/release/cmux-app}"`
   - `CMUX_CLI="${2:-$REPO_ROOT/target/release/cmux}"`
   - `CMUXD_REMOTE="${3:-$REPO_ROOT/daemon/remote/cmuxd-remote}"`
5. Extract version: `VERSION=$(grep '^version' "$REPO_ROOT/Cargo.toml" | head -1 | sed 's/.*"\(.*\)"/\1/')`
6. Verify all three binaries exist
7. Set `OUTPUT_DIR="${REPO_ROOT}/dist"` and `mkdir -p "$OUTPUT_DIR"`
8. Create temporary build directory structure:
```bash
BUILD_DIR=$(mktemp -d)
trap 'rm -rf "$BUILD_DIR"' EXIT

STAGING="$BUILD_DIR/SOURCES"
mkdir -p "$STAGING/icons"
mkdir -p "$BUILD_DIR/rpmbuild"/{BUILD,RPMS,SOURCES,SPECS,SRPMS}
```

9. Stage all files into SOURCES directory (flat structure for spec %install to consume):
```bash
# Binaries
cp "$CMUX_APP" "$STAGING/cmux-app"
cp "$CMUX_CLI" "$STAGING/cmux"
cp "$CMUXD_REMOTE" "$STAGING/cmuxd-remote"

# Desktop metadata
cp "$REPO_ROOT/packaging/desktop/com.cmux_lx.terminal.desktop" "$STAGING/"
cp "$REPO_ROOT/packaging/desktop/com.cmux_lx.terminal.metainfo.xml" "$STAGING/"

# Icons (flatten to simple names for spec)
cp "$REPO_ROOT/packaging/icons/hicolor/48x48/apps/com.cmux_lx.terminal.png" "$STAGING/icons/48x48.png"
cp "$REPO_ROOT/packaging/icons/hicolor/128x128/apps/com.cmux_lx.terminal.png" "$STAGING/icons/128x128.png"
cp "$REPO_ROOT/packaging/icons/hicolor/256x256/apps/com.cmux_lx.terminal.png" "$STAGING/icons/256x256.png"

# Completions
cp "$REPO_ROOT/packaging/completions/cmux.bash" "$STAGING/"
cp "$REPO_ROOT/packaging/completions/_cmux" "$STAGING/"
cp "$REPO_ROOT/packaging/completions/cmux.fish" "$STAGING/"

# Man page (gzipped)
gzip -9n < "$REPO_ROOT/packaging/man/cmux.1" > "$STAGING/cmux.1.gz"
```

10. Build the RPM:
```bash
rpmbuild -bb \
    --define "_cmux_version ${VERSION}" \
    --define "_sourcedir ${STAGING}" \
    --define "_topdir ${BUILD_DIR}/rpmbuild" \
    "$REPO_ROOT/packaging/rpm/cmux.spec"
```

11. Copy output to dist/:
```bash
RPM_FILE=$(find "$BUILD_DIR/rpmbuild/RPMS" -name "*.rpm" | head -1)
if [[ -z "$RPM_FILE" ]]; then
    echo "ERROR: rpmbuild produced no output" >&2
    exit 1
fi
cp "$RPM_FILE" "$OUTPUT_DIR/"
echo "Built: $OUTPUT_DIR/$(basename "$RPM_FILE")"
```

12. `chmod +x` the script.

  <action>
Create `packaging/scripts/validate-rpm.sh` that validates a built .rpm file.

Usage: `bash packaging/scripts/validate-rpm.sh [path-to-rpm]`
Default path: `dist/cmux-*.rpm` (glob for latest)

Script structure:
1. Shebang: `#!/usr/bin/env bash` + `set -euo pipefail`
2. SCRIPT_DIR/REPO_ROOT pattern
3. Check rpm command available: `command -v rpm &>/dev/null || { echo "ERROR: rpm not found. Install with: sudo apt install rpm" >&2; exit 1; }`
4. Accept optional arg for .rpm path, default to `$REPO_ROOT/dist/cmux-*.rpm` (use glob with single match check)
5. Track PASS/FAIL with helper function (same pattern as validate-deb.sh):
```bash
PASS=0 FAIL=0
check() {
    local desc="$1" cmd="$2"
    if eval "$cmd" &>/dev/null; then
        echo "  PASS: $desc"
        ((PASS++))
    else
        echo "  FAIL: $desc"
        ((FAIL++))
    fi
}
```
