  <action>
Create `packaging/scripts/build-deb.sh` that assembles a .deb package from pre-built binaries (per D-03, D-04).

Script structure (per D-01, using dpkg-deb):
1. Shebang and safety: `#!/usr/bin/env bash` + `set -euo pipefail`
2. SCRIPT_DIR and REPO_ROOT using existing pattern from detect-deps.sh
3. Parse optional arguments: positional args for binary paths (per D-04):
   - `CMUX_APP="${1:-$REPO_ROOT/target/release/cmux-app}"`
   - `CMUX_CLI="${2:-$REPO_ROOT/target/release/cmux}"`
   - `CMUXD_REMOTE="${3:-$REPO_ROOT/daemon/remote/cmuxd-remote}"`
4. Extract version: `VERSION=$(grep '^version' "$REPO_ROOT/Cargo.toml" | head -1 | sed 's/.*"\(.*\)"/\1/')`
5. Verify all three binaries exist with `[[ -f "$bin" ]]` loop, exit 1 with error if missing
6. Set `OUTPUT_DIR="${REPO_ROOT}/dist"` and `mkdir -p "$OUTPUT_DIR"`
7. Create staging directory: `PKG_ROOT=$(mktemp -d)` with trap to clean up on exit
8. Assemble staging directory tree (per D-05, D-06, D-07):

Binaries (per D-05):
```
install -Dm0755 "$CMUX_APP" "$PKG_ROOT/usr/bin/cmux-app"
install -Dm0755 "$CMUX_CLI" "$PKG_ROOT/usr/bin/cmux"
install -Dm0755 "$CMUXD_REMOTE" "$PKG_ROOT/usr/lib/cmux/cmuxd-remote"
```

Desktop metadata (per D-06):
```
install -Dm0644 "$REPO_ROOT/packaging/desktop/com.cmux_lx.terminal.desktop" "$PKG_ROOT/usr/share/applications/com.cmux_lx.terminal.desktop"
install -Dm0644 "$REPO_ROOT/packaging/desktop/com.cmux_lx.terminal.metainfo.xml" "$PKG_ROOT/usr/share/metainfo/com.cmux_lx.terminal.metainfo.xml"
```

Icons (per D-06):
```
for size in 48x48 128x128 256x256; do
    install -Dm0644 "$REPO_ROOT/packaging/icons/hicolor/${size}/apps/com.cmux_lx.terminal.png" \
        "$PKG_ROOT/usr/share/icons/hicolor/${size}/apps/com.cmux_lx.terminal.png"
done
```

Completions (per D-07):
```
install -Dm0644 "$REPO_ROOT/packaging/completions/cmux.bash" "$PKG_ROOT/usr/share/bash-completion/completions/cmux"
install -Dm0644 "$REPO_ROOT/packaging/completions/_cmux" "$PKG_ROOT/usr/share/zsh/vendor-completions/_cmux"
install -Dm0644 "$REPO_ROOT/packaging/completions/cmux.fish" "$PKG_ROOT/usr/share/fish/vendor_completions.d/cmux.fish"
```

Man page (gzipped, per D-07):
```
mkdir -p "$PKG_ROOT/usr/share/man/man1"
gzip -9n < "$REPO_ROOT/packaging/man/cmux.1" > "$PKG_ROOT/usr/share/man/man1/cmux.1.gz"
```

9. Create DEBIAN/control (per D-08, D-09, D-11):
```
mkdir -p "$PKG_ROOT/DEBIAN"
cat > "$PKG_ROOT/DEBIAN/control" << CTRL
Package: cmux
Version: ${VERSION}
Architecture: amd64
Maintainer: cmux <noreply@cmux.dev>
Section: x11
Priority: optional
Depends: libgtk-4-1, libfontconfig1, libfreetype6, libonig5, libgl1, libegl1, libharfbuzz0b, libglib2.0-0, libcairo2, libpango-1.0-0, libpangocairo-1.0-0, libpangoft2-1.0-0, libepoxy0, libxkbcommon0, libgraphene-1.0-0
Homepage: https://cmux.dev
Description: GPU-accelerated terminal multiplexer
 cmux provides tabs, splits, workspaces, and socket CLI control
 powered by Ghostty's GPU-accelerated terminal rendering.
CTRL
```

Note: Using non-t64 package names for Ubuntu 22.04 compatibility (per Research pitfall 3). `libglib2.0-0` is a virtual package on 24.04 provided by `libglib2.0-0t64`.

10. Build the .deb:
```
DEB_FILE="$OUTPUT_DIR/cmux_${VERSION}_amd64.deb"
dpkg-deb --build --root-owner-group "$PKG_ROOT" "$DEB_FILE"
echo "Built: $DEB_FILE"
```

  <action>
Create `packaging/scripts/validate-deb.sh` that validates a built .deb file.

Usage: `bash packaging/scripts/validate-deb.sh [path-to-deb]`
Default path: `dist/cmux_*_amd64.deb` (glob for latest)

Script structure:
1. Shebang: `#!/usr/bin/env bash` + `set -euo pipefail`
2. SCRIPT_DIR/REPO_ROOT pattern
3. Accept optional arg for .deb path, default to `$REPO_ROOT/dist/cmux_*_amd64.deb`
4. Track PASS/FAIL count with a helper function:
```bash
PASS=0 FAIL=0
check() {
    local desc="$1" cmd="$2"
    if eval "$cmd" &>/dev/null; then
        echo "  PASS: $desc"
        ((PASS++))
    else
        echo "  FAIL: $desc"
        ((FAIL++))
    fi
}
```

5. Validate these properties using `dpkg-deb` commands:

- `./usr/lib/cmux/cmuxd-remote` exists in listing

- `./usr/share/fish/vendor_completions.d/cmux.fish` exists
- `./usr/share/man/man1/cmux.1.gz` exists

**Metadata checks** (DEB-01, DEB-02) using `dpkg-deb -f "$DEB"`:
- `Package: cmux` in control output
