# Phase 9 — UI Design Contract

> Visual and interaction contract for Phase 9: UI Buttons and Menus. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | GTK4 CssProvider (APP_CSS in main.rs) |
| Preset | not applicable (native GTK4 app) |
| Component library | gtk4-rs 0.10 |
| Icon library | GTK4 symbolic icons (icon-name based) |
| Font | System default (GTK4 inherits from desktop theme) |

---

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, button margins |
| sm | 8px | Button internal padding, sidebar row padding (top/bottom), header bar element gaps |
| md | 16px | Sidebar row padding (left/right), section padding, popover menu internal padding |
| lg | 24px | Dialog padding |
| xl | 32px | Empty state padding |

Exceptions: Header bar buttons use 4px padding + 2px margin (matching established browser-nav-btn pattern). Touch target minimum 28x28px for all clickable elements.

| Role | Value | Usage |
|------|-------|-------|

| Secondary (30%) | #242424 | Sidebar, header bar, popover menu background, browser nav bar |
| Accent (10%) | #5b8dd9 | Active workspace row, active pane border, hovered divider, accent text in menus |
| Destructive | #cc4444 | Close Workspace confirmation, Close Pane (when last pane) |
| Attention | #e8a444 | Bell attention dot, reconnecting state |

| Text primary | #cccccc | All labels, button text, menu item text |

Accent reserved for: active workspace row background, active pane border, hovered divider, "Go" button text in browser nav, DevTools toggle checked state, accelerator hint text in menus.

Source: All values from existing APP_CSS. No new colors introduced.

## Interaction Contracts

### Header Bar (D-04, D-05, D-06)

- GTK4 `HeaderBar` replaces default window titlebar
- Left pack: [+] New Workspace, [Browser] New Browser
- Right pack: [Split H] [Split V] [Sidebar] [Hamburger]
- Each button: 28x28px min size, 4px padding, icon-name based symbolic icons
- Button hover: `rgba(255, 255, 255, 0.08)` background
- Button active: `rgba(255, 255, 255, 0.12)` background

**Icon assignments (Claude's Discretion):**

| Button | Icon Name | Tooltip |
|--------|-----------|---------|
| New Workspace | `tab-new-symbolic` | "New Workspace (Ctrl+N)" |
| New Browser | `web-browser-symbolic` | "New Browser (Ctrl+Shift+B)" |
| Split Horizontal | `view-dual-symbolic` | "Split Right (Ctrl+D)" |
| Split Vertical | `object-flip-vertical-symbolic` | "Split Down (Ctrl+Shift+D)" |
| Toggle Sidebar | `sidebar-show-symbolic` | "Toggle Sidebar (Ctrl+B)" |
| Hamburger | `open-menu-symbolic` | "Menu" |

### Sidebar Add Button (D-01)

- Position: Bottom of sidebar, below workspace list, full-width
- Label: "+" text (16px, weight 400, color #cccccc)
- Height: 36px (matches workspace row)
- Hover: #2e2e2e background
- Action: Creates new local workspace (same as Ctrl+N)

### Sidebar Close Button (D-02)

- Position: Right edge of each sidebar workspace row
- Visibility: Hidden by default, shown on row hover (CSS `:hover` on row reveals button)
- Label: Unicode multiplication sign (U+00D7)
- Size: 20x20px
- Color: #888888, hover #cccccc
- Action: Close workspace (with confirmation if workspace has multiple panes)

### Sidebar Context Menu (D-03)

- Trigger: Right-click (`GestureClick` button 3) on sidebar workspace row
- Menu model: `gio::Menu` with `gtk4::PopoverMenu`
- Items with accelerator hints:

| Item | Accelerator | Action |
|------|-------------|--------|
| Rename | Ctrl+Shift+R | Start inline rename |
| Close | Ctrl+Shift+W | Close workspace |
| Split Right | Ctrl+D | Split active pane right |
| Split Down | Ctrl+Shift+D | Split active pane down |

### Terminal Context Menu (D-08)

- Trigger: Right-click on terminal pane GLArea
- Items with accelerator hints:

| Item | Accelerator | Action |
|------|-------------|--------|
| Copy | (Ctrl+Shift+C) | Copy terminal selection |
| Paste | (Ctrl+Shift+V) | Paste from clipboard |
| --- | --- | separator |
| Split Right | Ctrl+D | Split this pane right |
| Split Down | Ctrl+Shift+D | Split this pane down |
| Close Pane | Ctrl+Shift+X | Close this pane |
| --- | --- | separator |
| Open Browser Here | Ctrl+Shift+B | Open browser in split |

### Browser Context Menu (D-09)

- Trigger: Right-click on browser preview pane
- Items:

| Item | Accelerator | Action |
|------|-------------|--------|
| Close Pane | Ctrl+Shift+X | Close browser pane |
| Open in External Browser | (none) | xdg-open URL |
| Copy URL | (none) | Copy current URL to clipboard |

### Hamburger Menu (D-11, D-12)

- Trigger: Click hamburger button in header bar
- Widget: `gtk4::MenuButton` with `gtk4::PopoverMenu` from `gio::Menu` model
- Sections (using `gio::Menu::new()` section model):

**File section:**

| Item | Accelerator |
|------|-------------|
| New Workspace | Ctrl+N |
| New SSH Workspace | Ctrl+Shift+S |
| New Browser | Ctrl+Shift+B |
| Close Pane | Ctrl+Shift+X |
| Close Workspace | Ctrl+Shift+W |
| Quit | Ctrl+Q |

**Edit section:**

| Item | Accelerator |
|------|-------------|
| Copy | Ctrl+Shift+C |
| Paste | Ctrl+Shift+V |
| Find | Ctrl+F |
| Preferences | (none) |

**View section:**

| Item | Accelerator |
|------|-------------|
| Toggle Sidebar | Ctrl+B |
| Split Right | Ctrl+D |
| Split Down | Ctrl+Shift+D |

**Help section:**

| Item | Accelerator |
|------|-------------|
| Keyboard Shortcuts | (none) |
| About cmux | (none) |

### Preferences (D-13)

- Action: Spawn `$EDITOR` (or `xdg-open`) on `~/.config/cmux/config.toml`
- Fallback: `xdg-open` if `$EDITOR` is not set
- No in-app settings UI

### Keyboard Shortcuts Window (D-14)

- Widget: `gtk4::ShortcutsWindow`
- Groups:

| Group | Shortcuts |
|-------|-----------|
| Workspaces | New, Close, Next, Previous, Rename, Switch 1-9 |
| Panes | Split Right, Split Down, Close, Focus Left/Right/Up/Down |
| Browser | Open, Close |
| SSH | New SSH Workspace |
| General | Toggle Sidebar, Copy, Paste, Find, Quit |

### About Dialog (D-15)

- Widget: `gtk4::AboutDialog`
- Program name: "cmux"
- Version: read from Cargo.toml at build time (`env!("CARGO_PKG_VERSION")`)
- License: project license
- Website: project URL
- Comments: "GPU-accelerated terminal multiplexer for Linux"

### Config.toml UI Section (D-16)

New config fields under `[ui.header_bar]`:

```toml
[ui.header_bar]
style = "gtk"           # "gtk" | "custom" | "none"
buttons_left = ["new_workspace", "browser"]
buttons_right = ["split_right", "split_down", "toggle_sidebar"]
```

Default: `style = "gtk"` with all buttons shown. Requires app restart.

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | "New Workspace" (sidebar + button tooltip) |
| Empty state heading | Not applicable (sidebar always has at least one workspace) |
| Empty state body | Not applicable |
| Error state (Preferences) | "Could not open config file. Create it at ~/.config/cmux/config.toml" |
| Error state (External browser) | "Could not open URL in external browser" |
| Destructive: Close Workspace | Shown only if workspace has unsaved state. Confirmation: "Close workspace '{name}'? All panes will be terminated." |
| Destructive: Close Pane | No confirmation for single pane close (matches terminal emulator convention) |
| Destructive: Quit | No confirmation (session persistence handles state) |
| About dialog tagline | "GPU-accelerated terminal multiplexer for Linux" |
| Tooltip format | "{Action Name} ({Shortcut})" e.g. "New Workspace (Ctrl+N)" |

## State Handling

| State | Visual Treatment |
|-------|-----------------|
| Header bar hidden (config: none) | No header bar rendered; window uses default GTK titlebar |
| Sidebar hidden | Sidebar add button and close buttons not visible; header bar remains |
| No browser installed | "New Browser" button still shown; error handled by existing browser.rs |
| Menu item disabled | Color #555555, not clickable (e.g., Copy when no selection) |
