**Test 1: Socket connectivity (SOCK-01, SOCK-03)**
```bash
python3 -c "
import sys, os
sys.path.insert(0, 'tests_v2')
os.environ['CMUX_SOCKET'] = os.path.join(os.environ.get('XDG_RUNTIME_DIR', f'/run/user/{os.getuid()}'), 'cmux', 'cmux.sock')
import cmux
c = cmux.Client()
print('ping:', c.ping())
print('identify:', c.identify())
"
```
Expected: ping returns True or {pong: true}; identify returns platform: "linux".

**Test 2: Workspace control (SOCK-02)**
```bash
python3 -c "
import sys, os
sys.path.insert(0, 'tests_v2')
os.environ['CMUX_SOCKET'] = os.path.join(os.environ.get('XDG_RUNTIME_DIR', f'/run/user/{os.getuid()}'), 'cmux', 'cmux.sock')
import cmux
c = cmux.Client()
print('workspaces:', c.workspace_list())
ws_id = c.workspace_create()
print('created:', ws_id)
c.workspace_rename(ws_id, 'test-ws')
print('renamed, list:', c.workspace_list())
c.workspace_close(ws_id)
print('closed, list:', c.workspace_list())
"
```
Expected: Creates a workspace, renames it, closes it. No crashes, no focus steal.
