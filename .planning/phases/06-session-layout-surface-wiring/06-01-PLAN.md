**3. Update `to_data()` Leaf arm to capture CWD via `/proc/{pid}/cwd` (per D-11):**

The Ghostty surface has no direct "get child PID" API. Instead, use the surface pointer and the Linux `/proc/self/fd` approach. The most reliable method: look up the surface in SURFACE_REGISTRY to find which pane it belongs to, then use `ghostty_surface_child_process_info` if available, or fall back to scanning `/proc` for the child shell PID.

Simpler approach per D-11: If ghostty exposes no PID API, the surface was created by `create_surface()` which launches a shell. The child PID can be obtained by `ghostty_inspector_*` or by tracking it. For now, implement a best-effort fallback:

```rust
SplitNode::Leaf { pane_id, uuid, surface, .. } => {
    let cwd = get_surface_cwd(*surface);
    let shell = std::env::var("SHELL").unwrap_or_else(|_| "/bin/sh".to_string());
    SplitNodeData::Leaf {
        pane_id: *pane_id,
        surface_uuid: *uuid,
        shell,
        cwd,
    }
}
```

Add a helper function `get_surface_cwd(surface: ffi::ghostty_surface_t) -> String` that:
- If surface is null, return `String::new()`
- Try to get the foreground process PID from the surface's PTY. Use `ghostty_surface_pty_fd` if available in FFI, then `tcgetpgrp()` on the PTY fd to get the foreground process group, then `readlink("/proc/{pid}/cwd")`.
- If no FFI function for PTY fd exists, fall back to: iterate `/proc/` entries looking for processes whose parent PID is the cmux process, or simply return `std::env::var("HOME").unwrap_or_default()` as fallback per D-11.
- The function should never panic -- always fall back to empty string or $HOME.

**1. Update `trigger_session_save()` in `src/app_state.rs` (line 435-460) per D-01, D-02, D-04:**

Replace the dummy leaf layout with real tree serialization. The current code at lines 447-452 creates a fake `SplitNodeData::Leaf` with pane_id=0 and nil UUID. Replace that with:

```rust
pub fn trigger_session_save(&self) {
    if let Some(ref notify) = self.save_notify {
        if let Some(ref tx) = self.session_tx {
            let session = crate::session::SessionData {
                version: 2,  // D-01: bump to version 2
                active_index: self.active_index,
                workspaces: self.workspaces.iter().enumerate().map(|(i, ws)| {
                    // D-02: save full split tree for ALL workspaces
                    let layout = if i < self.split_engines.len() {
                        self.split_engines[i].root.to_data()
                    } else {
                        // Fallback: shouldn't happen, but be safe
                        crate::split_engine::SplitNodeData::Leaf {
                            pane_id: 0,
                            surface_uuid: uuid::Uuid::nil(),
                            shell: String::new(),
                            cwd: String::new(),
                        }
                    };
                    // D-04: save active_pane_uuid per workspace
                    let active_pane_uuid = if i < self.split_engines.len() {
                        self.split_engines[i].active_pane_uuid()
                    } else {
                        None
                    };
                    crate::session::WorkspaceSession {
                        uuid: ws.uuid.to_string(),
                        name: ws.name.clone(),
                        active_pane_uuid,
                        layout,
                    }
                }).collect(),
            };
            let _ = tx.send(session);
        }
        notify.notify_one();
    }
}
```

Also add to `SplitNode`:

```rust
/// Find the UUID for a pane by pane_id. Returns None if not found.
pub fn find_uuid_for_pane(&self, target_id: u64) -> Option<String> {
    match self {
        SplitNode::Leaf { pane_id, uuid, .. } => {
            if *pane_id == target_id { Some(uuid.to_string()) } else { None }
        }
        SplitNode::Split { start, end, .. } => {
            start.find_uuid_for_pane(target_id)
                .or_else(|| end.find_uuid_for_pane(target_id))
        }
    }
}
```
