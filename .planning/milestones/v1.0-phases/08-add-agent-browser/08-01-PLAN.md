2. Create `src/browser.rs` with `BrowserManager` struct:

   ```rust
   use std::path::PathBuf;
   use std::process::{Child, Command};
   use serde_json::Value;

   /// Session name for the agent-browser daemon (one daemon per cmux instance).
   const SESSION_NAME: &str = "cmux";

   /// Preview pane state tracked by BrowserManager.
   #[derive(Debug, Clone, PartialEq, Eq)]
   pub enum PreviewState {
       Empty,
       Loading,
       Connected,
       Streaming,
       Error(String),
   }

   pub struct BrowserManager {
       daemon_process: Option<Child>,
       session_name: String,
       stream_task: Option<tokio::task::JoinHandle<()>>,
       pub frame_tx: Option<tokio::sync::mpsc::UnboundedSender<Vec<u8>>>,
       pub preview_state: PreviewState,
   }
   ```

   Implement these methods on BrowserManager:

   a. `pub fn new() -> Self` — initializes with session_name = SESSION_NAME, all fields None/Empty.

   b. `fn agent_browser_socket_dir() -> PathBuf` — mirrors agent-browser/cli/src/connection.rs logic:
      - Check `AGENT_BROWSER_SOCKET_DIR` env var (skip empty)
      - Check `XDG_RUNTIME_DIR` env var, append "agent-browser" (skip empty)
      - Fall back to `$HOME/.agent-browser`
      - Last resort: `std::env::temp_dir().join("agent-browser")`

   c. `pub fn daemon_socket_path(&self) -> PathBuf` — returns `{socket_dir}/{session_name}.sock`

   d. `pub fn stream_port_path(&self) -> PathBuf` — returns `{socket_dir}/{session_name}.stream`

   e. `fn daemon_ready(&self) -> bool` — tries `std::os::unix::net::UnixStream::connect` to daemon socket path. Returns true if connects, false otherwise.

   f. `pub fn ensure_daemon(&mut self) -> Result<(), String>` — per D-05 auto-start:
      - If `daemon_ready()` returns true, return Ok (daemon already running).
      - Find agent-browser binary: check `$PATH` for "agent-browser", then check alongside cmux binary at `std::env::current_exe()/../agent-browser`.
      - If not found, return `Err("agent-browser not found in PATH. Install it or place it alongside the cmux binary.".into())`.
      - Create socket dir if needed.
      - Spawn: `Command::new(binary_path).env("AGENT_BROWSER_DAEMON", "1").env("AGENT_BROWSER_SESSION", &self.session_name).env("AGENT_BROWSER_STREAM_PORT", "0").stdin(Stdio::null()).stdout(Stdio::null()).stderr(Stdio::piped()).spawn()`.
      - Store child in `self.daemon_process`.
      - Poll `daemon_ready()` with 200ms intervals, up to 50 retries (10 seconds total). If not ready after retries, return `Err("agent-browser daemon failed to start within 10 seconds".into())`.
      - Set `self.preview_state = PreviewState::Connected`.
      - Return Ok.

   g. `pub fn send_command(&self, action: &str, params: Value) -> Result<Value, String>` — send newline-delimited JSON to daemon socket:
      - Connect to `daemon_socket_path()` via `UnixStream::connect`.
      - Build request: `{"id": "cmux-{random}", "action": action, ...params}` (flatten params into request).
      - Write JSON + newline.
      - Read response line with `BufReader::read_line`.
      - Parse as `serde_json::Value`, return it.

   h. `pub fn read_stream_port(&self) -> Result<u16, String>` — read `stream_port_path()` file, parse as u16.

   i. `pub fn shutdown(&mut self)` — send `{"id": "cmux-shutdown", "action": "close"}` to daemon socket (ignore errors). If `daemon_process` is Some, wait 2 seconds then kill. If `stream_task` is Some, abort it.

3. Add `mod browser;` to src/main.rs module declarations (after `mod config;`).

4. Add `pub browser_manager: Option<browser::BrowserManager>` field to `AppState` in src/app_state.rs. Initialize it as `None` in `AppState::new()`.
